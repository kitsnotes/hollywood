소개

qpdfview는 탭 문서 뷰어입니다. PDF 지원에는 Poppler, PS 지원에는 libspectre, DjVu 지원에는 DjVuLibre, 인쇄 지원에는 CUPS, 인터페이스에는 Qt 툴킷을 사용합니다.

명령줄 옵션 및 환경설정 파일에 대한 자세한 내용은 수동 페이지를 참조하십시오.

기본 메뉴 ↑

파일 ↑

  • 열기... 현재 탭에서 열 파일을 선택할 수있는 대화 상자를 엽니 다. 탭이 없으면 파일이 새 탭에서 열립니다.
  • 새 탭에서 열기... 새 탭에서 열 파일을 선택할 수 있는 대화 상자를 엽니다.
  • 최근에 사용됨 최근에 사용한 파일 목록을 포함합니다(설정에서 활성화된 경우).
  • 새로고침 현재 탭에 표시된 파일을 다시 불러오기합니다.
  • 저장 주석 및 양식 필드의 변경 내용을 현재 탭에 표시된 파일에 저장합니다.
  • 다른이름으로 저장... 주석 및 양식 필드의 변경사항을 포함하여 현재 탭에 표시된 파일을 저장할 대화상자를 엽니다.
  • 사본 저장... 대화상자를 열어 현재 탭에 표시된 파일의 업그레이드된 사본을 저장합니다.
  • 인쇄... 대화상자를 열어 현재 탭에 표시된 파일을 인쇄합니다.

  • 주석 추가 이 옵션을 선택하면 프로그램에서 프레임이 드래그하여 주석을 추가할 때까지 기다립니다.

편집 도구 모음의 텍스트 필드에 원하는 페이지를 입력하고 Ctrl+K를 사용하여 반환 키를 눌러 현재 페이지를 변경할 수 있습니다.

기본 보기 ↑

백스페이스/스페이스 키는 이전/다음 페이지로 건너뜁니다. 홈/엔드 키가 첫 번째/마지막 페이지로 건너뜁니다. 화살표, 페이지 위로 및 페이지 아래로 키를 누르면 보임창이 스크롤됩니다. 컨트롤과 리턴을 누르면 마지막 이동 전 페이지로 이동합니다.

마우스 휠을 사용하여 위아래로 스크롤합니다. 마우스 휠을 사용하는 동안 ALT를 누르고 있으면 왼쪽과 오른쪽으로 스크롤됩니다. 마우스 휠을 사용하는 동안 CTRL을 누르고 있으면 문서의 크기가 조정됩니. 마우스 휠을 사용하는 동안 SHIFT를 누르면 문서가 회전합니다. (설정 대화 상자에서 키보드 수정도구를 변경할 수 있습니다.) 중간 마우스 버튼을 클릭하고 길게 눌러 확대/축소할 사각형을 선택합니다.

링크는 빨간색 프레임으로 강조 표시됩니다(설정에서 활성화된 경우). 링크를 마우스 왼쪽 단추로 클릭하여 연결된 페이지를 표시하거나 기본 브라우저에서 연결된 URL을 엽니다(설정에서 활성화된 경우). 링크를 가운데 클릭하여 링크된 페이지를 새 탭에 표시합니다. 링크 위를 맴돌면 해당 대상이 도구 팁으로 표시됩니다.

양식 필드는 파란색 프레임으로 강조 표시됩니다(설정에서 활성화된 경우). 양식 필드를 왼쪽 클릭하여 내용을 편집합니다. 양식 필드 위로 마우스를 가져가면 이름이 도구 설명으로 표시됩니다.

문서의 위치 소스는 첫 번째 일치하는 SyncTeX 노드의 항목을 선택적으로 포함하는 상황에 맞는 메뉴를 사용하여 소스 편집기에서 열 수 있습니다.

텍스트나 이미지를 복사하려면, Shift 키를 누르고 왼쪽 마우스 버튼을 클릭한 채로 복사할 영역 주위에 프레임을 끕니다. 마우스 버튼이 해제되면 텍스트 또는 영상을 클립보드로 복사할지 여부를 선택하는 팝업 메뉴가 표시됩니다.

주석 추가를 하려면 <CTRL 키를 누른 상태에서 왼쪽 마우스 버튼을 클릭한 채로 주석을 추가할 영역 주위에 프레임을 끌어다 놓습니다. 마우스 버튼이 해제되면 팝업 메뉴가 표시되어 텍스트를 추가하거나 주석을 강조 표시할 것인지 선택합니다. 주석 편집을 하려면 마우스 왼쪽 버튼을 클릭합니다. 주석 제거를 하려면 마우스 오른쪽 버튼을 클릭합니다. (주석을 추가하거나 제거하려면 Poppler 라이브러리 버전 0.20.1 이상이 필요합니다.) 주석 위로 마우스를 가져가면 해당 내용이 도구 설명으로 표시됩니다.

선택 항목을 확대하려면 SHIFT와 CTRL 키를 누른 상태에서 마우스 왼쪽 버튼을 클릭한 상태로 클릭하여 확대할 영역 주위로 프레임을 드래그합니다.

도구 및 메뉴 표시 줄의 표시여부는 기본 값이 "Shift+Alt+T"와 "Shift+Alt+M"인 구성 가능한 키보드 단축키를 사용하여 전환할 수 있습니다.

보조도구 보기 ↑

개요 보기 문서의 내용에 대한 개요가 표시됩니다. 항목을 왼쪽 클릭한 경우 기본 보기의 각 페이지로 이동합니다. 컨텍스트 메뉴를 사용하여 선택한 항목 아래의 모든 항목을 확장/축소할 수 있습니다.

개요 보기는 키보드와 마우스 휠을 사용하여 탐색할 수도 있습니다. Shift 키를 누른 상태에서 Up/Down 키를 사용하면 보기가 스크롤됩니다. Shift 키를 누른 상태에서 왼쪽/오른쪽 키를 사용하면 선택한 항목 아래에 있는 한 수준의 항목이 축소/확대됩니다. 제어를 누르고 왼쪽/오른쪽 키를 사용하면 선택한 항목과 해당 하위 항목이 완전히 축소/확대됩니다. Shift 키를 누른 상태에서 마우스 휠을 사용하면 선택한 항목이 변경됩니다. 컨트롤을 누르고 마우스 휠을 사용하면 선택한 항목이 확장/축소됩니다.

속성 보기는 문서에 포함된 추가 정보가 표시됩니다.

썸네일 보기는 문서의 페이지의 작은 버전을 표시합니다. 항목을 마우스 왼쪽 단추로 클릭하면 기본 보기의 해당 페이지로 이동합니다.

북마크 보기는 현재 탭에 표시된 문서와 관련된 북마크가 표시됩니다. 컨텍스트 메뉴에는 이전/다음 북마크로 이동하거나 북마크를 추가하는 항목이 포함되어 있습니다. 각 항목의 컨텍스트 메뉴에는 북마크를 편집하고 삭제하는 항목도 포함되어 있습니다.

설정에서 활성화된 경우, 확장 검색 도크는 모든 탭에서 수행되는 모든 검색 결과를 포함하는 개요를 표시합니다. 검색어를 강조하면서 결과를 둘러싼 텍스트도 표시합니다. 결과를 클릭하면 해당 탭이 열리고 해당 위치로 이동합니다.

바로가기 설정 ↑

바로 가기 설정을 통해 키보드 바로 가기를 개인 취향에 맞게 조정할 수 있습니다. 바로 가기를 편집하려면 바로 가기를 두 번 클릭하고 새 바로 가기를 입력하면 바로 가기가 변경됩니다. 각 작업은 세미콜론으로 구분 된 목록을 입력하여 여러 단축키를 가질 수 있습니다. 단일 문자가 아닌 키의 이름은 명시 적으로 작성되며 (예 : "Alt", "Ctrl", "PgDown") 일치 항목은 대소 문자를 구분하지 않습니다. 바로 가기에 한정자 키가 포함되어 있는 경우 키 이름이 더하기 기호로 조인되고 "Ctrl+A" 또는 "Ctrl++"와 같은 수정도구 키가 먼저 나열되어야 합니다.
