Introdução

qpdfview é um visualizador de documentos com abas. Utiliza o Poppler para suporte a PDF, libspectre para suporte a PS, DjVuLibre para suporte a DjVu, CUPS para suporte a impressão e QT Toolkit para sua interface.

Por favor, consulte a página do manual para obter informações sobre opções de linha de comando e arquivos de configuração.

Menu principal ↑

Arquivo ↑

  • Abrir... Abre uma janela para selecionar um arquivo para abrir na aba atual. Se não houver nenhuma aba, o arquivo é aberto é uma nova aba.
  • Abrir em nova aba... Abre uma janela para selecionar arquivos a serem abertos em novas abas.
  • Recentemente usado Contem uma lista dos arquivos recentemente utilizados (se ativado nas configurações).
  • Atualizar Recarrega o arquivo exibido na aba atual.
  • Salvar Salva as alterações nas anotações e nos campos do formulário no arquivo exibido na guia atual.
  • Salvar como... Abre um diálogo para salvar o arquivo exibido na aba atual incluindo alterações nas anotações e no formato.
  • Salvar cópia ... Abre uma caixa de diálogo para salvar uma cópia não modificada do arquivo exibido na guia atual.
  • Imprimir... Abre um diálogo para imprimir o arquivo exibido na aba atual.
  • Sair Fecha todas as abas e encerra o programa.

Editar↑

  • Próxima/Anterior/Primeira/Última página Pula para a próxima/anterior/primeira/última página do documento.
  • Ir para a página... Abre um diálogo para colocar uma página para ser exibida na aba atual.
  • Pular para trás/para frente Pula para trás/frente no histórico da página em exibição.
  • Pesquisar ... Abre a área de pesquisa abaixo da tela principal. A pesquisa inicia automaticamente dois segundos após a última alteração do termo de pesquisa ou é iniciada manualmente pressionando a tecla Enter. Se o modificador Shift for mantido pressionado, a pesquisa será iniciada em todas as guias, em vez de apenas na guia atual. (Esse efeito é revertido se a estação de pesquisa estendida estiver ativada nas configurações.)
  • Encontrar anterior/próximo Pula para a ocorrência anterior/próxima do termo procurado no documento.
  • Cancelar pesquisa Cancela a pesquisa em execução no momento e fecha a estação de pesquisa.
  • Copiar para a área de transferência Se marcado, o programa aguardará o arrastamento de um quadro para copiar textos ou imagens.
  • Adicionar anotação Se marcado, programa irá esperar um tempo para arrastar para adicionar anotações.
  • Opções... Abre o diálogo de configuração. O botão chamado 'Padrão' irá reverter todas as configurações em todas as tabelas para seus valores padrões. O botão chamado 'Padrão na tabela atual' irá reverter apenas as configurações na tabela atual para seus valores padrões.

A página atual pode ser alterada inserindo a página desejada no campo de texto na barra de ferramentas de edição, que pode ser focada usando Ctrl + K e pressionando a tecla Enter.

Visualização↑

  • Contínuo Determina se todas as linhas do layout da página estão visíveis ao mesmo tempo.
  • Duas páginas Determina se o layout da página usa duas colunas.
  • Duas páginas com capa Determina se o layout da página usa duas colunas com uma quebra após a capa.
  • Várias páginas Determina se o layout da página usa várias colunas. O número de colunas pode ser alterado na caixa de diálogo de configurações.
  • Direita para a esquerda Determina como as páginas são organizadas em layouts com várias páginas por linha.
  • Ampliar Aumenta o fator de escala em 10%. O fator de escala máximo é de 5000%.
  • Diminuir o zoom Diminui o fator de escala em 10%. O fator de escala mínimo é 10%.
  • Tamanho original Redefine o fator de escala para 100%.
  • Ajustar à largura da página Determina o fator de escala para que todas as páginas se ajustem totalmente à vista principal horizontalmente.
  • Ajustar ao tamanho da página Determina o fator de escala de modo que todas as páginas se ajustem totalmente à vista principal horizontal e verticalmente.
  • Girar para a esquerda Gira as páginas para a esquerda/anti-horário em 90 graus.
  • Girar para a direita Gira as páginas para a direita/no sentido horário em 90 graus.
  • Inverter cores Determina se as páginas são renderizadas com cores invertidas.
  • Converter em escala de cinza Determina se as páginas são convertidas em escala de cinza após a renderização.
  • Aparar margens Permite a remoção de margens vazias da página após a renderização.
  • Composição Ativa a composição da página renderizada com a cor do papel para ajustar o fundo dos documentos digitalizados.
  • Barras de ferramentas Este menu contém uma lista de barras de ferramentas disponíveis para acessar ações usadas com freqüência.
  • Boxes Este menu contém uma lista de boxes disponíveis para a exibição de informações suplementares.
  • Fontes ... Abre uma caixa de diálogo listando as fontes usadas no documento.
  • Tela cheia Alterna o modo de tela cheia, ou seja, a janela é mostrada maximizada e sem decoração.
  • Apresentação ... Entra no modo de apresentação que pode ser deixado pressionando F12 ou escapando. A tecla inicial / final é usada para exibir a primeira / última página. As teclas de seta para a direita e para baixo, o espaço e a tecla de página para baixo pulam para a próxima página. As teclas de seta para a esquerda e para cima, o backspace e a tecla page up saltam para a página anterior. Pressionar controle e retornar retorna à página antes do último salto.

O fator de escala pode ser alterado digitando a porcentagem desejada no campo de texto na barra de ferramentas da visualização, que pode ser focada usando Ctrl + L e pressionando a tecla Enter.

  • Fechar guia Fecha a guia atual.
  • Fechar todas as guias Fecha todas as guias.

As nove primeiras guias também podem ser ativadas pressionando o modificador Alt e pressionando as teclas de uma a nove.

  • Remover todos os favoritos Remove todos os favoritos.

Menu principal ↑

A tecla de espaço pula para a página anterior/seguinte. A tecla Home/End pula para a primeira /última página. As teclas de seta, página acima e página abaixo rolam a janela de exibição. Pressionar controle e retornar salta para a página antes do último salto.

Usando a roda do mouse para cima e para baixo. Segurar ALT enquanto usa a roda do mouse rola para a esquerda e para a direita. Segurar CTRL enquanto usa a roda do mouse redimensiona o documento. Segurar SHIFT enquanto usa a roda do mouse gira o documento. (Os modificadores do teclado podem ser alterados na caixa de diálogo de configurações.) Clique e segure o botão do meio do mouse para selecionar um retângulo para ampliar.

Os links são realçados pelas bordas em vermelhos (se ativado nas configurações). Clique com o botão esquerdo do mouse em um link para mostrar a página vinculada ou abra o URL vinculado no navegador padrão (se ativado nas configurações). Clique com o botão direito do mouse em um link para mostrar a página vinculada em uma nova guia. Passar o mouse sobre um link exibe seu destino como uma dica de ferramenta.

Os campos do formulário são realçados por quadros azuis (se ativado nas configurações). Clique com o botão esquerdo do mouse em um campo de formulário para editar seu conteúdo. Passar o mouse sobre um campo de formulário exibe seu nome como uma dica de ferramenta.

A fonte de uma posição no documento pode ser aberta no editor de fontes usando o menu de contexto, que opcionalmente contém um item para o primeiro nó correspondente do SyncTeX .

Para copiar textos ou imagens , mantenha pressionado SHIFT , clique e mantenha pressionado o botão esquerdo do mouse para arrastar um quadro ao redor da área a ser copiada. Depois que o botão do mouse é liberado, um menu pop-up é exibido para selecionar se deseja copiar textos ou imagens para a área de transferência.

Para adicionar anotações , mantenha pressionado CTRL , clique e segure o botão esquerdo do mouse para arrastar um quadro pela região a ser anotada. Depois que o botão do mouse é liberado, um menu pop-up é exibido para selecionar se você deseja adicionar texto ou realçar anotações. Clique com o botão esquerdo para editar anotações . Clique com o botão direito do mouse para remover anotações . (Adicionar e remover anotações requer a versão 0.20.1 ou superior da biblioteca Poppler.) Passar o mouse sobre uma anotação exibe seu conteúdo como uma dica de ferramenta.

Para aumentar o zoom em uma seleção , mantenha pressionados SHIFT e CTRL , clique e mantenha pressionado o botão esquerdo do mouse para arrastar um quadro ao redor da região para aumentar o zoom.

A visibilidade da ferramenta e das barras de menus pode ser alternada usando atalhos de teclado configuráveis cujos valores padrão são "Shift + Alt + T" e "Shift + Alt + M".

Visualizações auxiliares ↑

A visão de tópicos exibe uma visão geral do conteúdo do documento. Um clique esquerdo em uma entrada pula para a respectiva página na visualização principal. O menu de contexto pode ser usado para expandir / recolher todos os itens abaixo do item selecionado.

A visualização de estrutura de tópicos também pode ser navegada usando o teclado e a roda do mouse. Pressionar Shift e usar as teclas Para cima / Para baixo rolará a exibição. Pressionar Shift e usar as teclas Esquerda / Direita recolherá / expandirá um nível de itens abaixo do item selecionado. Manter o Control pressionado e usar as teclas Esquerda / Direita recolherá / expandirá completamente o item selecionado e seus filhos. Manter a tecla Shift pressionada e usar a roda do mouse mudará o item selecionado. Manter o controle e usar a roda do mouse expandirá / reduzirá o item selecionado.

A visão de propriedades exibe informações adicionais contidas no documento.

A exibição de miniaturas exibe versões menores das páginas do documento. Um clique esquerdo em uma entrada pula para a respectiva página na visualização principal.

A exibição de favoritos mostra os favoritos relacionados ao documento exibido na guia atual. O menu de contexto contém entradas para ir para a marca anterior / seguinte ou adicionar marcadores. O menu de contexto de cada entrada também contém entradas para editar e excluir marcadores.

Se ativado nas configurações, o box de pesquisa estendido mostra uma visão geral contendo os resultados de todas as pesquisas realizadas em todas as guias. Também mostra o texto em torno de um resultado com o termo de pesquisa sendo enfatizado. Clicar em um resultado abre a guia correspondente e salta para sua posição.

Configurações ↑

Configurações de comportamento ↑

  • Abrir URL Determina se os links para URL são abertos no navegador padrão.
  • Atualização automática Determina se os arquivos são recarregados automaticamente se alterados no disco.
  • Rastrear usado recentemente Se ativado, um menu usado recentemente está disponível e lista quantos arquivos usados recentemente, conforme definido na opção "Contagem usada recentemente". (Só terá efeito depois de reiniciar.)
  • Manter fechado recentemente Se ativado, um menu fechado recentemente está disponível e lista quantas guias fechadas recentemente como definido na opção "Contagem fechada recentemente". Manter as guias disponíveis pode aumentar consideravelmente o uso de memória. (Efetivo após o reinício.)
  • Restaurar guias Se ativada, as guias e suas propriedades são salvas ao finalizar o programa e restauradas na reinicialização.
  • Restaurar favoritos Se ativado, os favoritos são salvos ao finalizar o programa e restaurados na reinicialização.
  • Restaurar configurações por arquivo Se ativado, as configurações por arquivo são salvas / restauradas nas abas de abertura / fechamento.
  • Salvar intervalo do banco de dados O intervalo após o qual as guias, marcadores ou configurações por arquivo são salvas no banco de dados.
  • Sincronizar apresentação Se ativada, a tela principal pode ser usada para controlar a tela de apresentação.
  • Tela de apresentação O índice da tela para exibir a exibição da apresentação.
  • Sincronizar exibição de estrutura de tópicos Se ativada, a exibição de estrutura de tópicos destacará a primeira entrada apontando para a página atual sempre que a página atual for alterada.
  • Relative jumps If enable, the view port will keep its position relative to the visible page when jumping to another page.
  • Rolagem mínima Se ativada, a porta de visualização será rolada apenas se a posição de destino ainda não estiver visível.
  • Fator de zoom O número pelo qual o fator de escala é multiplicado e dividido após o zoom.
  • Cor do destaque Determina a cor dos realces.
  • Duração do destaque Determina a duração dos destaques temporários.
  • Cor da anotação A cor das anotações adicionadas pelo programa usando palavras-chave em cores SVG ou notação hexadecimal '#rrggbb'.
  • Editor de fontes O programa que é iniciado para editar fontes. '% 1' é substituído pelo caminho absoluto do arquivo. '% 2' resp. '% 3' é substituído pela linha e número da coluna

Configurações gráficas ↑

  • Usar lado a lado As páginas serão divididas em lado a lado antes da renderização e do cache para reduzir o uso de memória.
  • Manter pixmaps obsoletos Se ativado, após a alteração do fator de zoom, os pixmaps armazenados em cache serão usados para desenhar aproximações dos pixmaps finais até que estejam disponíveis ao custo de maior uso de memória.
  • Usar taxa de pixels do dispositivo Se ativada, a maior taxa de pixels do dispositivo disponível é usada para renderizar no físico e exibir em resolução lógica.
  • Use logical DPI If enabled, the horizontal and vertical resolution of the monitor as reported by the operating system is used to define a scale factor of 100%.
  • Decore páginas Determina se as páginas são separadas do plano de fundo.
  • Decore links Determina se os links são realçados por bordas em vermelho.
  • Decore os campos do formulário Determina se os campos do formulário são realçados por bordas em azul.
  • Cor do plano de fundo Determina a cor do plano de fundo usando palavras-chave em cores SVG ou notação hexadecimal '#rrggbb'.
  • Cor do papel Determina a cor das páginas usando palavras-chave em cores SVG ou notação hexadecimal '#rrggbb'.
  • Cor do plano de fundo da apresentação Substitui a cor do papel no cálculo da cor de plano de fundo da visualização da apresentação e é definida usando palavras-chave em cores SVG ou a notação hexadecimal '#rrggbb'.
  • Páginas por linha Número de páginas por linha no modo de layout 'várias páginas'.
  • Espaçamento entre páginas Determina o espaço vertical e horizontal exibido entre as páginas.
  • Espaçamento entre miniaturas Determina o espaço vertical e horizontal exibido entre as miniaturas.
  • Tamanho da miniatura Determina o tamanho das miniaturas.
  • Tamanho do cache O programa tentará armazenar em cache as páginas renderizadas dentro deste limite, aumentando a velocidade de renderização, com o custo de maior uso de memória.
  • Pré-busca O programa tentará renderizar páginas que ainda não são visíveis, aumentando a velocidade de renderização, com o custo do aumento do uso do cache.
  • Distância de pré-busca O número de linhas além da linha atual e o dobro do número de linhas antes da linha atual que é pré-buscada.
  • Configurações do PDF
    • Antialiasing Melhore a qualidade da exibição ao custo do desempenho
    • Antialiasing de texto Melhora a qualidade da exibição do texto ao custo do desempenho
    • Sugestão de texto Pode melhorar a qualidade da exibição do texto para determinadas fontes.
    • Pré-impressão de superimposição Se ativado, uma pré-visualização da impressão de uma camada de cores sobre a outra é exibida.
    • Modo de linha fina Seleciona a otimização usada para desenhar linhas finas.
    • Back-end Seleciona o back-end usado para renderizar páginas.
  • Configurações do PS
    • Gráficos antialias bits Determina a quantidade de antialiasing para gráficos.
    • Bits de antialias de texto Determina a quantidade de antialiasing de texto.

Configurações de atalho ↑

As configurações de atalho permitem o ajuste dos atalhos do teclado de acordo com as preferências pessoais. Os atalhos são alterados clicando duas vezes no atalho para editá-lo e inserindo o novo atalho. Cada ação pode ter vários atalhos, inserindo uma lista separada por ponto e vírgula. Os nomes das chaves que não são letras únicas são gravados explicitamente (por exemplo, "Alt", "Ctrl", "PgDown"), onde a correspondência não diferencia maiúsculas de minúsculas. Se o atalho contiver teclas modificadoras, os nomes das teclas serão unidos por um sinal de adição e as teclas modificadoras deverão ser listadas primeiro, por exemplo "Ctrl + A" ou "Ctrl ++".

  • Girar Os modificadores do teclado para girar usando a roda do mouse na vista principal.
  • Rolagem Os modificadores do teclado para rolar horizontalmente usando a roda do mouse na tela principal.
  • Copiar para a área de transferência Os modificadores do teclado para copiar textos ou imagens de uma página para a área de transferência.
  • Adicionar anotação Os modificadores do teclado para adicionar anotações a uma página.
  • Zoom na seleção Os modificadores do teclado para aumentar o zoom em uma seleção em uma página.
